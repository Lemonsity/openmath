Sigma Algebra
A family of subsets \( \mathcal{ F } \subseteq X \) is said to be a \( \sigma \)-algebra on \( X \) whenever
  • \( \emptyset , X \in \mathcal{ F } \)
  • If \( A \in \mathcal{ F } \) then \( X \setminus A \in \mathcal{ F } \)
  • If \( \left( A _ n \right) \) is a sequence of sets in \( \mathcal{ F } \) then
    • \[ \bigcup _ { n = 1 } ^ \infty A _ n \in \mathcal{ F } \]
Measurable Space
An ordererd pair \( \left( X, \mathcal{ F } \right) \) consisting of a set \( X \) and a sigma algebra on \( X \) is called a measurable space
Measurable With Resepect to a Sigma Algebra
Suppose that \( \left( X, \mathcal{ F } \right) \) is a measure space, then any set \( A \in \mathcal{ F } \) is said to be an \( \mathcal{ F } \)-measurable set.

Whenever the sigma algebra is implicitly known we simplify the above and just say that \( A \) is measurable.

Measure
A measure is an extended real-valued function \(\mu\) defined on a \(\sigma\)-algebra \( \mathcal{ F } \) of subsets of \(X\) such that
  • \(\mu(\emptyset)=0\)
  • \(\mu(E) \ge 0\) for all \(E \in \mathcal{ F } \)
  • \(\mu\) is countably additive in the sense that if \(\left(E_n\right)\) is any disjoint sequence of sets in \( \mathcal{ F } \), then
    • \[ \mu\left(\bigcup_{n=1}^{\infty} E_n\right)=\sum_{n=1}^{\infty} \mu\left(E_n\right) . \]

Note that we say that a measure is finite or that it is \( \sigma \)-finite when the the measure does not take on infinity in the above definition.

Measurable Function
Let \((X, \mathcal{F})\) be a measurable space, where \(\mathcal{F}\) is a \(\sigma\)-algebra of subsets of \(X\), and let \((Y, \mathcal{G})\) be another measurable space. A function \(f: X \to Y\) is called measurable if for every set \(B \in \mathcal{G}\) we have: \[ f^{-1}(B) \in \mathcal{F} \quad \text{for all } B \in \mathcal{G}. \]
Almost Everywhere Convergence
Let \((X, \mathcal{F}, \mu)\) be a measure space, and let \(\{f_n\}_{n=1}^{\infty}\) be a sequence of measurable functions \(f_n: X \to \mathbb{R}\) (or \(\mathbb{C}\)). We say that \(\{f_n\}\) converges to \(f\) almost everywhere if there exists a set \(N \subset X\) with \(\mu(N) = 0\) such that \[ \lim_{n \to \infty} f_n(x) = f(x) \quad \text{for all } x \in X \setminus N. \]

In other words, the sequence \(\{f_n(x)\}\) converges to \(f(x)\) at all points \(x\) except possibly on a set \(N\) where the measure \(\mu(N)\) is zero. This means that the convergence holds "almost everywhere" in \(X\), despite there being potentially a small (measure-zero) set where it may fail to converge. Note that almost everywhere is often abbreviated as a.e.

ideas to be organized into the above

The notation \( L_\mu^2(X) \) represents the **space of square-integrable functions** on a measure space \( (X, \mathcal{F}, \mu) \). Which are functions \( f: X \rightarrow \mathbb{C} \) (or \( \mathbb{R} \)) such that: \[ \int_X |f(x)|^2 \, d\mu < \infty. \] This means that the squared absolute value of \( f \) is integrable with respect to the measure \( \mu \).

The differential \( d\mu \) in an integral \( \int_X f \, d\mu \) indicates that the integration is taken with respect to the measure \( \mu \). The integral \( \int_X f \, d\mu \) is defined as a limit (or sum) over measurable "slices" of \( X \) rather than just points. Formally, for a non-negative measurable function \( f: X \rightarrow [0, \infty] \), the integral with respect to \( d\mu \) is defined as \[ \int_X f \, d\mu = \sup \left\{ \sum_{i=1}^n f(x_i) \, \mu(A_i) : \{A_i\}_{i=1}^n \text{ is a finite measurable partition of } X \right\}. \] This expression takes a supremum over all possible finite partitions of \( X \) into measurable sets \( A_i \) with \( x_i \in A_i \) and \( \sum_{i=1}^n f(x_i) \, \mu(A_i) \) providing an approximation of the integral over each slice weighted by \( \mu(A_i) \).
