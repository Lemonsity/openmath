Metric
A metric on a set \( X \) is a function \( d : X \times X \to \mathbb{ R } ^ { \ge 0 } \) such that:
  • \( d \left( x, y \right) = 0 \) iff \( x = y \)
  • \( d \left( x, y \right) = d \left( y, x \right) \)
  • The triangle inequality holds:
    • \[ d \left( x, z \right) \le d \left( x, y \right) + d \left( y, z \right) \]
Ball in a Metric
Suppose that \( d \) is a metric then we define the ball of radius \( r \in \mathbb{ R } ^ { \gt 0 } \) around a point \( p \in X \) \[ B \left( p, r \right) = \left\{ x \in X : d \left( p, x \right) \lt r \right\} \]

Before looking further lets see some examples of metrics

  • \( \mathbb{ R } ^ n \): \( d \left( x, y \right) = \sqrt{ \sum _ { i = 1 } ^ n \left( x _ i - y - i \right) ^ 2 } \) with this metric a ball is a circle
  • \( \mathbb{ R } ^ n \): \( d \left( x, y \right) = \sum _ { i = 1 } ^ n \left\lvert x _ i - y _ i \right\rvert \), the so called manhattan distance because in manhattan you have to go along the streets locking you onto a grid, this makes balls into diamonds
  • \( \mathbb{ R } ^ n \): \( d \left( x, y \right) = \max _ { i = n } ^ n \left( \left\lvert x _ i - y _ i \right\rvert \right) \), the maximum distance in any direction, a ball here pushes in each dimension to get to the radius, inducing a ball to be a square

Metric Topology Induced by a Metric \( d \)

All Metric Topologies Are T2
All metric topologies are \( T_2 \)
Suppose that \( X \) is a metric topology induced by the metric \( d \) and that \( x_1, x_2 \in X \) then if we consider \( r = \frac{ d \left( x_1, x_2 \right) }{ 2 } \) then we know that \( B \left( x_1, r \right) \cap B \left( x_2, r \right) = \emptyset \) for if it was not then there is a point \( p \) in their intersection such that \( d \left( x_1, p \right) \lt r \) and that \( d \left( x_2, p \right) \lt r \). But then also by the triangle inequality we have that \( d \left( x_1, x_2 \right) \le d \left( x_1, p \right) + d \left( p, x_2 \right) \) so that \[ d \left( x_1, x_2 \right) - d \left( x_1, p \right) \le d \left( p, x_2 \right) \] but then \( d \left( x_1, x_2 \right) - d \left( x_1, p \right) \gt 2 r - r = r \) so we conclude \( d \left( p, x_2 \right) \ge r \) which is a contradiction, so the intersection must be empty.

Note that this shows that the metric topologies are a subset of the T2 topologies, since we know not every topology is T2, then we know that not every topology is metrizable.

The Max of Two Metrics Is a Metric
Suppose that \( d_1, d_2 \) are metrics on \( X \) then \( d = \max \left( d_1, d_2 \right) \) is also a metric

Firstly note that for any \( x, y \in X \) we have that \( d _ 1 \left( x, y \right) \le d \left( x, y \right) \) and also \( d _ 2 \left( x, y \right) \le d \left( x, y \right) \) therefore since \( d _ 1, d _ 2 \) both map into \( \mathbb{ R } ^ { \ge 0 } \) so does \( d \).

If \( x = y \) then \( d _ 1 \left( x, y \right) = d _ 2 \left( x, y \right) = 0 \) so that \( d \left( x, y \right) = \max \left( 0, 0 \right) = 0 \), for the other direction we work with the contraposive so assume that \( x \neq y \) and then we'll prove that \( d \left( x, y \right) \neq 0 \) but note that \( d _ 1 \left( x, y \right) \gt 0 \) and same with \( d _ 2 \left( x, y \right) \gt 0 \) because they are metrics, and therefore the max of the two is also \( \gt 0 \) so we are done.

Note \( d \left( x, y \right) = \max \left( d _ 1 \left( x, y \right), d _ 2 \left( x, y \right) \right) = \max \left( d _ 1 \left( y, x \right), d _ 2 \left( y, x \right) \right) = d \left( y, x \right) \)

Now suppose that \( x, y, z \in X \) then \( d \left( x, z \right) \) is either equal to \( d _ 1 \left( x, z \right) \) or \( d _ 2 \left( x, z \right) \) suppose the latter, then \( d \left( x, z \right) = d _ 1 \left( x, z \right) \le d _ 1 \left( x, y \right) + d _ 1 \left( y, z \right) = d \left( x, y \right) + d \left( y, z \right) \) and in the other case it works for \( d _ 2 \) by the same argument.
