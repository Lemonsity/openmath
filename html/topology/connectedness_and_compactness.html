It is not path connected, for the sake of contradiction suppose that it was, then there would be path \( \gamma: \left[ 0, 1 \right] \to \bar S \) from \( \left( 0, 0 \right) \) to any point \( s \in S \), since the set \( \left\{ t : \gamma \left( t \right) \in 0 \times \left[ -1, 1 \right] \right\} \) is closed, then it has a largest element \( b \).

Now we can re-parameterize \( \gamma \) on \( \left[ b, 1 \right] \) to \( \gamma ^ \prime \) defined on \( \left[ 0, 1 \right] \).
