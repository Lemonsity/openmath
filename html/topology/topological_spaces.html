Topology on a set \( X \)

A topology on \( X \) is a collection \( \mathcal{T} \) of subsets of \( X \), that is: it is a subset of the \( P \left ( X \right ) \), with the following properties:

  • \( \emptyset , X \in \mathcal{T} \)
  • For any \( \mathcal{ A } \subseteq \mathcal{ T } \)
    • \[ \bigcup \mathcal{ A } \in \mathcal{ T } \]
  • For any \( \mathcal{ C } \subseteq \mathcal{ T } \) such that \( \mathcal{ C } \) is finite
    • \[ \bigcap \mathcal{ C } \in \mathcal{ T } \]
The set \( X \) along with \( \mathcal{T} \) satsifying the above conditions is called a topological space and is denoted by \( \left ( X , \mathcal{T} \right ) \)
Open set
Suppose \( \left ( X , \mathcal{T} \right ) \) is a topological space, if \( U \in \mathcal{T} \) then we say that \( U \) is open with respect to \( X \).
a set filled with open sets is open
Let \( X \) be a topological space, and \( A \subseteq X \). Suppose that for each \( x \in A \) there is an open set \( U \) containing \( x \) such that \( U \subseteq A \). Show that \( A \) is open in \( X \).

Since for each \( x \in A \), there is an open set \( U_{x} \) such that \( x \in U_{x} \subseteq A \), then \( A \) is covered by subsets, therefore it is a union, and we can write \( A = \bigcup_{x \in A} U_{x} \)

Since each \( U_{x} \) was assumed to be open with respect to \( X \), then an arbitrary union of them is also open with respect to \( X \), in other words \( A \) must be open.

The Countable Complement Topology
Let \( X \) be a set and define the set \[ CC = \left\{ U \subseteq X: \left\lvert X \setminus U \right\rvert \le \aleph_0 \right\} \cup \left\{ \emptyset \right\} \] where we've used \( \aleph_0 \) then \( CC \) is a topology and we denote it by \( \mathcal{ T } _ C \)

We'll note that \( X \in CC \) as \( X \setminus X = \emptyset \) is certainly finite, on the other hand \( \emptyset \in CC \) by definition

Now let \( \mathcal{ A } \subseteq CC \), then \[ X \setminus \bigcup \mathcal{ A } = X \setminus \bigcup _ { A \in \mathcal{ A } } A = \bigcap _ { A \in \mathcal{ A } } \left( X \setminus A \right) \] therefore we have an intersection of countable sets and therefore is countable

Now let \( \mathcal{ C } \subseteq CC \), where \( \mathcal{ C } \) is finite, then \[ X \setminus \bigcap \mathcal{ C } = X \setminus \bigcup _ { C \in \mathcal{ C } } C = \bigcup _ { C \in \mathcal{ C } } \left( X \setminus A \right) \] so we have a finite union of countable sets, and therefore it is countable. Therefore the set is closed under arbitrary unions and finite intersections and thus is a topology.

Note that \( \mathcal{T} = \{X\} \cup \{U \subset X : X - U \text{ is infinite}\}. \) is not a topology so long as \( X \) is infinite. For example pick some \( p \in X \) then every singleton \( \left\{ q \right\} \) where \( q \neq p \) is open in \( X \) because \( X \setminus \left\{ q \right\} \) is infinite, if it were to be a topology then we would know that \( \bigcup _ { q \neq p } \left\{ q \right\} = X \setminus \left\{ q \right\} \) is open, but \( X \setminus \left( X \setminus \left\{ q \right\} \right) = \left\{ q \right\} \) which is finite, thus a contradiction, so it cannot be a topology.

The Intersection of Topologies Is a Topology
If \( \mathfrak{T} \) is a collection of topologies then \( \bigcap \mathfrak{T} \) is a topology.
Since for any \( \mathcal{ T } \in \mathfrak{T} \) we know that \( \emptyset , X \in \mathcal{ T } \) then \( \emptyset , X \in \bigcap \mathfrak{T} \), then suppose that \( \mathcal{ A } \subseteq \bigcap \mathfrak{T} \) then for any \( \mathcal{ T } \in \mathfrak{T} \) we know \( \bigcup \mathcal{ A } \) is a union of elements in \( \mathcal{ T } \) so that \( \bigcup \mathcal{ A } \in \mathcal{ T } \), therefore \( \bigcup \mathcal{ A } \in \bigcup \mathfrak{T} \), similarly if \( \mathcal{ C } \subseteq \bigcap \mathfrak{T} \) is finite then for any \( \mathcal{ T } \in \mathfrak{ T } \) we know that \( \bigcap \mathcal{ C } \in \mathcal{ T } \) therefore \( \bigcap \mathcal{ C } \in \mathfrak{ T } \), therefore \( \bigcap \mathfrak{ T } \) is a topology.
The Intersection of a Collection of Sets That Are Supersets of a Given Set and Satisfy a Property Is the Smallest Set Which Satisfies the Property and Is Still a Superset of the Given Set
Let \( X \) be a set and \( Q \) a predicate then suppose that \( \mathcal{ C } \) is a collection of sets such that for any \( C \in \mathcal{ C } \) we have \( X \subseteq C \) and \( Q \left( C \right) \) then if \( Q \left( \bigcap \mathcal{ C } \right) \) holds true then it is the smallest such set where it holds true

Suppose that there were a smaller set, which is to say we had some set \( S \) where \( X \subseteq S \subset \bigcap \mathcal{ C } \) and \( Q \left( S \right) \), since \( X \subseteq S \) and \( Q \left( S \right) \) then \( X \in \mathcal{ C } \) therefore \( \bigcap \mathcal{ C } \subseteq S \), a contradiction, and thus there is no strictly smaller set.

Now we show that this set is unique, as if \( B \) is another set such that \( Q \left( B \right) \), \( X \subseteq B \) and \( B \subseteq \bigcap \mathcal{ C } \) then the first two imply that \( B \in \mathcal{ C } \) and therefore \( \bigcap \mathcal{ C } \subseteq B \), so that \( \mathcal{ C } = B \)

Given a Family of Topologies There Is a Unique Smallest Topology Containing All of Them
Suppose that \( \mathfrak{ T } \) is a collection of topologies on \( X \), and let \( \mathfrak{ F } \) be the collection of topologies such that for any \( \mathcal{ F } \in \mathfrak{ F } \), we know that for every \( \mathcal{ T } \in \mathfrak{ T } \) we have \( \mathcal{ T } \subseteq \mathcal{ F } \), then \[ \bigcap \mathfrak{ F } \] is the unique smallest topology on \( X \) that contains all the topologies in \( \mathfrak{ T } \)
We know \( \bigcap \mathfrak{ F } \) is a topology. Additionally it must be the smallest such topology that contains all topologies in \( \mathfrak{ T } \) for if there was another such one \( \mathcal{ Y } \) that did, then we would know that \( \mathcal{ Y } \in \mathfrak{ F } \) and therefore \( \bigcap \mathfrak{ F } \subseteq \mathcal{ Y } \) so that \( \bigcap \mathfrak{ F } = \mathcal{ Y } \) , note that the final equality shows that it is uniquely the smallest.
Given a Family of Topologies There Is a Unique Largest Topology Contained in Every Topology in the Family
Suppose that \( \mathfrak{ T } \) is a collection of topologies on, then \( \bigcap \mathfrak{ T } \) is the unique largest topology that is contained in \( \mathcal{ T } \) for each \( \mathcal{ T } \in \mathfrak{ T } \).

We know that it is a topology, to see that it is the largest, suppose that there was a larger topology \( \mathcal{ Y } \) that is contained in \( \mathcal{ T } \) for each \( \mathcal{ T } \in \mathfrak{ T } \) but this implies that \( \mathcal{ Y } \subseteq \bigcap \mathfrak{ T } \) which is a contradiction, so \( \bigcap \mathfrak{ T } \) is the largest.

It is unique as given another topology \( \mathcal{ Z } \) that is contained in each \( \mathcal{ T } \) then as above we know that \( \mathcal{ Z } \subseteq \bigcap \mathfrak{ T } \) but sinze \( \mathcal{ Z } \) is the largest topology with this property then we know that \( \bigcap \mathfrak{ T } \subseteq \mathcal{ Z } \) so that \( \bigcap \mathfrak{ T } = \mathcal{ Z } \) as needed.

Example of the Largest and Smallest Topology
With \( X = \left\{ a, b, c \right\} \), \( \mathcal{ T } _ 1 = \left\{ \emptyset , X, \left\{ a \right\}, \left\{ a, b \right\} \right\} \) and \( \mathcal{ T } _2 = \left\{ \emptyset , X, \left\{ a \right\}, \left\{ a, b \right\} \right\} \), find the smallest topology containing both, and the largest one contained in both.

By the above proposition we know the smallest topology is the intersection of all topologies that contain all open sets from \( \mathcal{ T } _1 \) and \( \mathcal{ T } _ 2 \) so any such topology contains at least \( \emptyset , X, \left\{ a \right\}, \left\{ a, b \right\}, \left\{ b, c \right\} \) due to the closure under unions and intersection, then each such topology contains at least \[ \left\{ \emptyset , X, \left\{ a \right\}, \left\{ b \right\}, \left\{ a, b \right\}, \left\{ b, c \right\} \right\} \] since the above is a topology, then the intersection of all such topologies must be the above set (the smallest possible set, is included in the intersection, thus it equals it).

The largest topology contained in both \( \mathcal{ T } _ 1, \mathcal{ T } _ 2 \) is \( \mathcal{ T } _ 1 \cap \mathcal{ T } _ 2 = \left\{ \emptyset , X, \left\{ a \right\} \right\} \)

The Power Set is a Topology
Given a set \( X \), then \( P \left( X \right) \) is a topology on it.

We note that \( \emptyset \subseteq X \) and \( X \subseteq X \) therefore we know that \( \emptyset , X \in P \left( X \right) \).

Now suppose that \( \mathcal{ A } \) is a collection of sets from \( P \left( X \right) \). Thus if we look at the union it is of the form \[ \bigcup _ { A \in \mathcal{ A } } A \] and since we know that each \( A \in P \left( X \right) \) then \( A \subseteq X \) and therefore it is a union of elements from \( X \) and so \( \bigcup _ { A \in \mathcal{ A } } A \subseteq X \) therefore \( \bigcup _ { A \in \mathcal{ A } } \in P \left( X \right) \)

Now suppose that \( \mathcal{ C } \) is a finite collection of elements from \( P \left( X \right) \), then similarly we find that \( \bigcap \mathcal{ C } \subseteq X \).

finer and coarser topologies

suppose that \( \mathcal{T} \) and \( \mathcal{T} ' \) are two topologies on a given set \( X \). If \( \mathcal{T} \subseteq \mathcal{T} ' \), then \( \mathcal{T} ' \) is finer than \( \mathcal{T} \). If the reverse inclusion is true, then we say that \( \mathcal{T} ' \) is coarser than \( \mathcal{T} \), there are also strict variations of these definitions for the strict inclusions.

comparable topologies

given two topologies, they are comparable if at least one is finer than the other
