Open Map
A function \( f : X \to Y \) is said to be an open map if for every open set \( U \in \mathcal{ T } _X \) the set \( f \left( U \right) \) is open in \( Y \)
The Projection Maps Are Open
The project maps: \( \pi _ X : X \times Y \to X \) and \( \pi _ Y : X \times Y \to Y \) are open

Let \( M \) be an open set in \( \mathcal{ T } _ { X \times Y } \) therefore it is of the form \( M = \bigcup \mathcal{ A } \) where \( \mathcal{ A } \subseteq \left\{ U \times V : U \in \mathcal{ T } _X , V \in \mathcal{ T } _Y \right\} \).

But then \( \pi_X \left( M \right) = \pi_X \left( \bigcup \mathcal{ A } \right) \) = \( \bigcup \left\{ \pi_X \left( U _ A \times V _ A \right) : U _ A \times V _ A \in \mathcal{ A } \right\} = \bigcup \left\{ U _A : U_ A \times V _A \in \mathcal{ A } \right\} \) so that \( \pi _ X \left( M \right) \) is a unions of open sets of \( \mathcal{ T } _X \) and therefore is open in \( X \). Symetrically wealso have that \( \pi _ Y \) is open.

Quotient Map
Given an \( \left( X, \sim \right) \), then we define the quotient map of \( \sim \) as a function \( \pi : X \to X \setminus \sim \) (where we are mapping into the space modded out by the quotient) \[ \pi \left( x \right) = \left[ x \right] \]
Quotient Topology by Universal Properties
Given a quotient map \( \pi : X \to X \setminus \sim \) then there exists a unique topology on \( X \setminus \sim \) such that
  • \( \pi \) is continuous
  • If \( f : X \setminus \sim \to Z \) is a function and \( f \circ h \) is continuous, then so is \( f \)
Specifically it is given by the set \[ \left\{ V \subseteq X \setminus \sim : \pi ^ { -1 } \left( V \right) \in \mathcal{ T } _ X \right\} \] We denote this topology by \[ \mathcal{ T } _ { X \setminus \sim } \]
First we show that it is a topology and then we show that it is unique.
Direction of a Vector Space
Suppose that \( V \) is a vector space then we define the direction of \( V \) by considering the equivalence relation induced by \( \left\{ \left( v_1, v _ 2 \right) : v_1 = \alpha v_2 , \alpha \in \mathbb{ R } ^ { \gt 0 } \right\} \) which we denote by \( D \) then we define \[ \operatorname{ dir } \left( V \right) = V \setminus D \]
The Direction of the Reals
What is the quotient topology of \( \operatorname{ dir } \left( \mathbb{ R } \right) \) ?

Before starting we can see that \( \operatorname{ dir } \left( \mathbb{ R } \right) = \left\{ \left[ -1 \right], \left[ 0 \right], \left[ 1 \right] \right\} \), now let's figure out the topology, we know that it is made up of sets whose inverse images under \( \pi : \mathbb{ R } \to \operatorname{ dir } \left( \mathbb{ R } \right) \) is open.

We note that \( \pi ^ { -1 } \left( \left[ -1 \right] \right) = \left( - \infty , 0 \right) \), that \( \pi ^ { -1 } \left( \left[ 0 \right] \right) = \left\{ 0 \right\} \) and that \( \pi ^ { -1 } \left( \left[ 1 \right] \right) = \left( 0, \infty \right) \). Any other subset of \( \operatorname{ dir } \left( \mathbb{ R } \right) \) is a union of these sets, moreover since the inverse image allows unions to pass through, it shows that the inverse image of every subset is also open, thus is the discrete topology.
