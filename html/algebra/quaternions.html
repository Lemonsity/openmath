The Quaternions
The collection of quaternions are: \[ \mathbb{ H } := \left\{ a + b \mathbf{ i } + c \mathbf{ j }+ d \mathbf{ k } \right\} \] where \( a, b, c, d \in \mathbb{ R } \) and \( \mathbf{ i } ^ 2 = \mathbf{ j } ^ 2 = \mathbf{ k } ^ 2 = \mathbf{ ijk }= -1 \)
Real Part of the Quaternion
Given a quaternion \(q = a + bi + cj + dk\), we say that \(a\) is the scalar part of the quaternion (or real part), and is denoted by \( r \left( q \right) \)
Vector Part of the Quaternion
Given a quaternion \(q = a + bi + cj + dk\), we say that \(bi + cj + dk\) is the vector part of the quaternion, and is denoted by \( v \left( q \right) \)

Note that we think of the vector part of a quaternion as an element of \( \mathbb{ R } ^ 3 \) , and so it inherits all the operations from that space, such as equality, as noted in the next corollary.

Conjugation Distributes
For any \( p, q \in \mathbb{ H } \) we have \[ p = q \iff \left( r\left( p \right) = r\left( q \right) \land v\left( p \right) = v\left( q \right) \right) \]
Suppose that \( p = a + bi + cj + dk \) and \( q = u + xi + yj + zk \) then \( p = q \) if and only if \( a = u \) , \( b = x \) , \( c = y \) , \( d = z \) which is true if and only if \( r\left( p \right) = r\left( q \right) \) and \( v\left( p \right) = v\left( q \right) \), as needed.
Quaternion Addition
\[ (a_1+b_1\mathbf i + c_1\mathbf j + d_1\mathbf k) + (a_2 + b_2\mathbf i + c_2\mathbf j + d_2\mathbf k) = (a_1 + a_2) + (b_1 + b_2)\mathbf i + (c_1 + c_2)\mathbf j + (d_1 + d_2) \mathbf k, \]
Conjugation is Additive
Suppose that \( p, q \in \mathbb{ H } \) then \[ \overline{ p + q } = \overline{ p } + \overline{ q } \]
Suppose that \( p = a + bi + cj + dk \) and \( q = u + xi + yj + zk \) then \[ \begin{align} \overline{ p + q } &= \overline{ \left( a + u \right) + \left( b + x \right) i + \left( c + y \right) j + \left( d + z \right) } \\ &= \left( a + u \right) - \left( b + x \right) i - \left( c + y \right) j - \left( d + z \right) \\ &= a - bi - cj - dk + u - xi - yj - zk \\ &= \overline{ p } + \overline{ q } \end{align} \]
Scalar Quaternion Multiplication
\[ \lambda(a + b\,\mathbf i + c\,\mathbf j + d\,\mathbf k) = \lambda a + (\lambda b)\,\mathbf i + (\lambda c)\,\mathbf j + (\lambda d)\,\mathbf k. \]
Identity Quaternion
\[ e := i + j + k \]
Conjugate Quaternion
Given the quaternion \( q = a + b \mathbf{ i }+ c \mathbf{ j } + d \mathbf{ k } \), it's conjugate is given by \[ \overline{ q } := a - b \mathbf{ i }- c \mathbf{ j } - d \mathbf{ k } \]
Vector part of the Conjugate is minus 1 Times the Original
For any \( p \in \mathbb{ H } \) we have \[ v \left( \overline{ p } \right) = -1 v \left( p \right) \]
Suppose that \( p = a + bi + cj + dk \) \[ v \left( \overline{ p } \right) = -bi + -cj + -dk = \left( -1 \right) v \left( p \right) \] as needed.
Real part of the Conjugate doesn't Change
For any \( p \in \mathbb{ H } \) we have \[ r \left( \overline{ p } \right) = r \left( p \right) \]
Suppose that \( p = a + bi + cj + dk \) \[ r \left( \overline{ p } \right) = a = r \left( p \right) \] as needed.
Conjugate Doesn't change the Real Part
For any \( p \in \mathbb{ H } \), such that \( v\left( p \right) = 0 \), then \[ \overline{ p } = p \]
Conjugate only Applies to Vector Part
For any \( p \in \mathbb{ H } \) we have \[ \overline{ p } = r\left( p \right) - v\left( p \right) \]
\[ \overline{ p } = \overline{ r \left( p \right) + v\left( p \right) } = r\left( p \right) - v\left( p \right) \]
Pure Quaternion
A quaternion \( p \in \mathbb{ H } \) is said to be pure, deiff \( r\left( p \right) = 0 \)
Product of Two Quaternions
Suppose that \( p, q \in \mathbb{ H } \), then we define \[ p q := r \left( p \right) r \left( q \right) - v\left( p \right) \cdot v\left( q \right) + r \left( p \right) v\left( q \right) + r\left( q \right) v\left( p \right) + v \left( p \right) \times v \left( q \right) \]
Product Commutes in the Real Part
For any \( p, q \in \mathbb{ H } \) we have \[ r \left( pq \right) = r\left( qp \right) \]
\[ r\left( pq \right) = r \left( p \right) r \left( q \right) - v\left( p \right) \cdot v\left( q \right) = r \left( q \right) r \left( p \right) - v\left( q \right) \cdot v\left( p \right) = r\left( qp \right) \]
Dot Product of Two Quaternions
Suppose that \( p, q \in \mathbb{H} \), then we define \[ p \cdot q := \operatorname{re}(p) \operatorname{re}(q) + \operatorname{v}(p) \cdot \operatorname{v}(q) \]
Conjugation is a Homomorphism in the Real Part
For any \( p, q \in \mathbb{ H } \) we have \[ r \left( \overline{ pq } \right) = r\left( \overline{ p } ~ \overline{ q } \right) \]
First of all, we know conjugation doesn't change the real part, so we have: \[ r \left( \overline{ pq } \right) = r\left( pq \right) = r \left( p \right) r \left( q \right) - v\left( p \right) \cdot v\left( q \right) \] and then we know that \[ r \left( \overline{ p } ~ \overline{ q } \right) = r \left( \overline{ p } \right) r \left( \overline{ q } \right) - v\left( \overline{ p } \right) \cdot v\left( \overline{ q } \right) \] Since minus comes out of the vector part, and constants can get pulled out of the dot product we have that \[ r \left( \overline{ p } \right) r \left( \overline{ q } \right) - v\left( \overline{ p } \right) \cdot v\left( \overline{ q } \right) = r\left( p \right) r \left( q \right) - \left( - 1 \right) \left( - 1 \right) v\left( p \right) \cdot v\left( q \right) = r\left( p \right) r \left( q \right) - v\left( p \right) \cdot v\left( q \right) \] therefore we have that \( r\left( \overline{ pq } \right) = r\left( \overline{ p } ~ \overline{ q } \right) \)
Conjugation Swaps Order in the Vector Part
For any \( p, q \in \mathbb{ H } \) we have \[ v \left( \overline{ pq } \right) = v\left( \overline{ q } ~ \overline{ p } \right) \]
We recall that, and so \[ v \left( \overline{ pq } \right) = v \left( r\left( pq \right) - v\left( pq \right) \right) = -v\left( pq \right) \] and then we know that \[ \begin{align} v\left( \overline{ q } ~ \overline{ p } \right) &= r \left( \overline{ q } \right) v\left( \overline{ p } \right) + r\left( \overline{ p } \right) v\left( \overline{ q } \right) + v \left( \overline{ q } \right) \times v \left( \overline{ p } \right) \\ &= - r \left( q \right) v\left( p \right) - r\left( p \right) v\left( q \right) + v \left( \overline{ q } \right) \times v \left( \overline{ p } \right) \\ &= - r \left( q \right) v\left( p \right) - r\left( p \right) v\left( q \right) + \left( -1 \right) \left( -1 \right) v \left( q \right) \times v \left( p \right) \\ &= - r\left( p \right) v\left( q \right) - r \left( q \right) v \left( p \right) - v\left( p \right) \times v \left( q \right) \\ &= - v\left( qp \right) \end{align} \] as needed.
Conjugation Distributes by Swapping
For any \( p, q \in \mathbb{ H } \) we have \[ \overline{ pq } = \overline{ q } ~ \overline{ p } \]
We show they are equal: \[ r\left( \overline{ pq } \right) = r\left( \overline{ p } ~ \overline{ q } \right) = r\left( \overline{ q } ~ \overline{ p } \right) \] and then we recall \( v\left( \overline{ pq } \right) = v\left( \overline{ q } ~ \overline{ p } \right) \), so that \( \overline{ pq } = \overline{ q } ~ \overline{ p } \)
